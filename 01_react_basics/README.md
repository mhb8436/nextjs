# React 기초 학습

이 프로젝트는 React의 기본 개념을 학습하기 위한 예제입니다.

## 프로젝트 생성 방법

```bash
# Vite CLI를 사용하여 React + TypeScript 프로젝트 생성
npm create vite@latest 01_react_basics -- --template react-ts

# 생성된 프로젝트로 이동
cd 01_react_basics

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 설치 방법

```bash
# 프로젝트 클론
git clone [repository-url]
cd 01_react_basics

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 학습 내용

- React 컴포넌트의 기본 구조
- JSX 문법
- 컴포넌트 렌더링
- 기본적인 이벤트 처리

## 주요 파일 구조

- `src/App.tsx`: 메인 애플리케이션 컴포넌트
- `src/main.tsx`: 애플리케이션 진입점
- `index.html`: HTML 템플릿

## 실행 방법

개발 서버가 실행되면 기본적으로 http://localhost:5173 에서 애플리케이션을 확인할 수 있습니다.
